# Homeowners' Subdivision Management

Utility apps for managing homeowners' subdivision.

### Basic Features

  - Homeowners Registry
  - Checklists
  - Homeowners' vehicles & stickers
  - Solicitation

### Basic Finance & Accounting Features

  - Cash Voucher
  - Purchase Order
  - Reimbursement
  - Item & Asset Registry
  - General Journal
  - Automated General Ledger
  - Automated Item Ledger
